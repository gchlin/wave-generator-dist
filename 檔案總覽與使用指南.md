## 📂 檔案列表

### 1️⃣ 核心檔案（必備）

#### `wave-generator-source.tar.gz` (73 KB)
**這是什麼？** 完整的專案源碼壓縮檔  
**用途：** 本地開發、修改程式碼  
**如何使用：**
```bash
# 解壓縮
tar -xzf wave-generator-source.tar.gz -C wave-generator

# 安裝依賴
cd wave-generator
npm install

# 啟動開發伺服器
npm run dev
```

#### `wave-generator-dist/` 資料夾
**這是什麼？** 已建置的靜態網頁  
**用途：** 直接部署到 GitHub Pages  
**如何使用：** 上傳資料夾**內的檔案**到 GitHub  

**重要提醒：** ⚠️ 上傳的是資料夾**裡面的檔案**，不是資料夾本身！

應該上傳：
```
✅ index.html
✅ assets/
   ✅ index-xxx.css
   ✅ index-xxx.js
```

不應該是：
```
❌ wave-generator-dist/
   └── index.html
   └── assets/
```
