# 波動圖形產生器 - 完整使用說明

## 📦 檔案說明

您會收到兩個檔案：

1. **wave-generator-source.tar.gz** - 完整專案源碼
2. **wave-generator-dist/** - 已建置的網頁檔案（可直接部署）

## 🚀 快速開始

### 方法一：直接使用已建置版本（最簡單）

1. 解壓縮後，直接將 `wave-generator-dist/` 資料夾上傳到 GitHub Pages
2. 或使用任何靜態網頁伺服器（如 Netlify、Vercel）

### 方法二：從源碼開始開發

1. 解壓縮 `wave-generator-source.tar.gz`
```bash
tar -xzf wave-generator-source.tar.gz -C wave-generator
cd wave-generator
```

2. 安裝依賴
```bash
npm install
```

3. 啟動開發伺服器
```bash
npm run dev
```

4. 在瀏覽器開啟 http://localhost:5173

## 📤 部署到 GitHub Pages

### 步驟詳解

1. **建立 GitHub Repository**
   - 在 GitHub 建立新的 repository，例如：`wave-generator`
   - 不要初始化 README

2. **修改配置**
   - 編輯 `vite.config.js`，將 `base` 改為你的 repository 名稱：
   ```javascript
   export default defineConfig({
     plugins: [react()],
     base: '/wave-generator/'  // 改成你的 repo 名稱
   })
   ```

3. **建置專案**
   ```bash
   npm run build
   ```

4. **部署到 GitHub Pages**
   ```bash
   cd dist
   git init
   git add -A
   git commit -m 'Initial deploy'
   git branch -M gh-pages
   git remote add origin https://github.com/你的使用者名稱/wave-generator.git
   git push -u origin gh-pages
   ```

5. **啟用 GitHub Pages**
   - 到 repository 設定
   - 找到 "Pages" 選項
   - Source 選擇 `gh-pages` 分支
   - 儲存後等待幾分鐘

6. **訪問網站**
   - 網址：`https://你的使用者名稱.github.io/wave-generator/`

### 自動化部署腳本

建立 `deploy.sh`：
```bash
#!/usr/bin/env sh
set -e

npm run build

cd dist

git init
git add -A
git commit -m 'deploy'
git push -f git@github.com:你的使用者名稱/wave-generator.git main:gh-pages

cd -
```

使用方式：
```bash
chmod +x deploy.sh
./deploy.sh
```

## 🎨 功能使用指南

### 基本功能

#### 1. 調整波形參數
- **振幅 (A)**：控制波的高度，範圍 0.1 - 3.0
- **波長 (λ)**：控制波的長度，範圍 0.5 - 4.0
- **相位 (φ)**：控制波的起始位置，範圍 0° - 360°

#### 2. 快速選擇預設場景
系統提供三組教學常用場景：

**基本波形**
- 標準正弦波：A=1, λ=2, φ=0°
- 大振幅：A=2, λ=2, φ=0°
- 小振幅：A=0.5, λ=2, φ=0°

**波長比較**
- 長波：A=1, λ=3, φ=0°
- 標準波長：A=1, λ=2, φ=0°
- 短波：A=1, λ=1, φ=0°

**相位差**
- 同相 (0°)：A=1, λ=2, φ=0°
- 相差 90°：A=1, λ=2, φ=90°
- 反相 (180°)：A=1, λ=2, φ=180°

#### 3. 配色方案選擇
- **投影模式**（推薦）：深藍色波形，適合課堂投影
- **黑白模式**：純黑白，適合列印講義
- **深色模式**：淺色波形配深色背景，適合夜間使用

#### 4. 顯示選項
- ☑ 顯示波長標註：在波形下方標示 λ
- ☑ 顯示振幅標註：在波峰標示 A
- ☐ 顯示網格：背景顯示參考網格

#### 5. 動畫功能
- 勾選「啟用動畫」可看到波的傳播效果
- 適合展示波動的時間演進

### 進階功能

#### 匯出圖形
1. **單一下載**：調整好參數後，點選「下載 SVG」
2. **批量下載**：
   - 點選「加入收藏」儲存多個設定
   - 在收藏區勾選想要的圖形
   - 點選「下載所選 (n)」批量下載

#### SVG 檔案特性
- ✅ 向量圖形，可無限縮放不失真
- ✅ 可用 Illustrator、Inkscape 編輯
- ✅ 可直接插入 PowerPoint、Word
- ✅ 檔案小，適合網頁使用

## 🛠 技術規格

### 波形函數
```
y(x, t) = A · sin(kx - ωt + φ)

其中：
- A：振幅
- k = 2π/λ：波數
- ω = 2πf：角頻率
- φ：相位（弧度）
- t：時間（動畫用）
```

### 系統需求
- Node.js 16 或以上（開發用）
- 現代瀏覽器（Chrome、Firefox、Safari、Edge）

### 專案技術棧
- React 18
- Vite 5
- 純 JavaScript（無額外依賴）

## 📝 常見問題

### Q: 如何修改預設參數？
A: 編輯 `src/utils/presets.js` 中的 `defaultParams`

### Q: 如何新增更多預設場景？
A: 編輯 `src/utils/presets.js` 中的 `teachingPresets`

### Q: 如何調整 SVG 輸出尺寸？
A: 在 `src/utils/waveGenerator.js` 中修改 `scale` 參數

### Q: 動畫太快/太慢怎麼辦？
A: 在 `src/utils/presets.js` 中調整 `frequency` 參數

### Q: 可以加入多波疊加功能嗎？
A: 可以！這是規劃中的功能，目前版本專注於單波展示
