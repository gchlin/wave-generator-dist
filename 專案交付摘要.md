# 波動圖形產生器 - 專案交付摘要

## 📦 交付內容

### 1. 專案源碼
**檔案**: `wave-generator-source.tar.gz`
- 完整的 React + Vite 專案
- 所有源代碼和配置檔案
- 不含 node_modules（需自行安裝）

### 2. 已建置版本
**資料夾**: `wave-generator-dist/`
- 已編譯的靜態網頁
- 可直接部署到任何靜態伺服器
- 包含 HTML、CSS、JavaScript

### 3. 說明文件
- **使用說明.md**: 完整使用手冊（Markdown 格式）
- **快速開始.html**: 互動式快速開始指南（HTML 格式）

## 🎯 專案特點

### 功能清單
✅ **核心功能**
- 即時調整波形參數（振幅 A、波長 λ、相位 φ）
- SVG 向量圖形生成
- 即時預覽波形變化
- 一鍵下載 SVG 檔案

✅ **進階功能**
- 動畫展示（波的傳播）
- 自動標註（波長、振幅）
- 多種配色方案（投影、黑白、深色）
- 快速預設場景（9 種教學常用設定）
- 收藏與批量下載

✅ **教學優化**
- 中文界面
- 抽象單位（不顯示實際數值）
- 投影片友善配色
- 適合高中物理程度

### 技術規格

**前端框架**: React 18
**建置工具**: Vite 5
**圖形格式**: SVG（可縮放向量圖形）
**數學模型**: y(x,t) = A·sin(kx - ωt + φ)

**瀏覽器支援**:
- Chrome / Edge（推薦）
- Firefox
- Safari

**部署方式**:
- GitHub Pages（推薦）
- Netlify
- Vercel
- 任何靜態網頁伺服器

## 🚀 快速開始三步驟

### 開發環境
```bash
# 1. 解壓縮並安裝
tar -xzf wave-generator-source.tar.gz -C wave-generator
cd wave-generator
npm install

# 2. 啟動開發伺服器
npm run dev

# 3. 在瀏覽器開啟
# http://localhost:5173
```

### 生產部署
```bash
# 1. 建置
npm run build

# 2. 部署到 GitHub Pages
cd dist
git init
git add -A
git commit -m 'deploy'
git push -f git@github.com:使用者名稱/wave-generator.git main:gh-pages
```

## 📊 預設場景列表

### 基本波形組
1. **標準正弦波**: A=1.0, λ=2.0, φ=0°
2. **大振幅**: A=2.0, λ=2.0, φ=0°
3. **小振幅**: A=0.5, λ=2.0, φ=0°

### 波長比較組
4. **長波**: A=1.0, λ=3.0, φ=0°
5. **標準波長**: A=1.0, λ=2.0, φ=0°
6. **短波**: A=1.0, λ=1.0, φ=0°

### 相位差組
7. **同相 (0°)**: A=1.0, λ=2.0, φ=0°
8. **相差 90°**: A=1.0, λ=2.0, φ=90°
9. **反相 (180°)**: A=1.0, λ=2.0, φ=180°

## 🔧 自訂與擴充

### 修改預設參數
編輯 `src/utils/presets.js`:
```javascript
export const defaultParams = {
  amplitude: 1,        // 預設振幅
  wavelength: 2,       // 預設波長
  phase: 0,            // 預設相位
  frequency: 0.5,      // 動畫速度
  // ...其他選項
};
```

### 新增預設場景
在 `src/utils/presets.js` 中的 `teachingPresets` 新增：
```javascript
'你的分類': [
  { name: '場景名稱', amplitude: 1.5, wavelength: 2.5, phase: 45 },
  // 更多場景...
]
```

### 調整顏色
編輯 `src/utils/colorSchemes.js` 自訂配色。

### 修改 SVG 尺寸
在 `src/utils/waveGenerator.js` 中調整 `scale` 參數：
```javascript
const scale = { x: 100, y: 80 };  // 調整這裡
```

## 🐛 已知限制

1. **動畫不會保存在 SVG**
   - 匯出的 SVG 是靜態圖形
   - 建議在下載前關閉動畫

2. **批量下載限制**
   - 瀏覽器可能會詢問是否允許多次下載
   - 檔案會依序下載，不會打包成 ZIP

3. **瀏覽器兼容性**
   - IE 不支援（需使用現代瀏覽器）
